🎨 RustSSH 色卡系统（Color System v3.0）

设计目标：组件驱动 + 层级统一 + 可直接映射代码

⸻

一、设计核心（必须理解）

1.1 三层结构（所有颜色来源）

Foundation（基础色）
→ Surface（层级色）
→ Semantic（语义色）

⸻

1.2 状态表达规则（全局统一）

Default → Surface_1
Hover   → Surface_2
Active  → Surface_3

⸻

1.3 Accent 使用规则

仅用于：

- 选中（Tab / Sidebar）
- Focus
- Cursor
- Primary Button

禁止：

- Hover 背景
- 大面积铺色（除 Primary）

⸻

七、实现建议（Rust / egui）

⸻

7.1 Token 结构

struct Theme {
    bg_primary: Color32,
    surface_1: Color32,
    surface_2: Color32,
    surface_3: Color32,
    text_primary: Color32,
    accent: Color32,
}

⸻

7.2 状态映射（关键）

fn surface(level: u8) -> Color32 {
    match level {
        1 => theme.surface_1,
        2 => theme.surface_2,
        3 => theme.surface_3,
        _ => theme.surface_1,
    }
}
