# UI 名词定义 (UI Terminology)

## 1. 顶栏区域 (Header Units)

- **顶栏 (Unified Header)**: 位于窗口最上方，高度固定为 **32px** 的全局交互容器。在 macOS 上采用沉浸式设计（透明底色、全尺寸内容视图）。
- **标签栏 (Tab Bar)**: 顶栏中用于容纳会话标签的水平区域，支持溢出滚动。
- **标签位 (Tab Item/Slot)**: 单个会话标签的物理区域，包含文字标题、背景 and 选中态指示条。默认宽度 140px，高度 32px。
- **操作组 (Action Group)**: 固定位置的功能按钮簇，包含 `+ (新建标签)` 和 `⚡ (快速连接)`。
- **控制组 (Control Group)**: 位于顶栏 right 侧，包含 `⚙ (设置)` 及非 Mac 平台的窗口控制按钮（最小化、最大化、关闭）。
- **物理分隔线 (Separator)**: 位于标签栏与操作组之间，高度为 14px-16px 的微弱垂直线，用于逻辑分层。

## 2. 内容与反馈 (Feedback & Interaction)

- **渐变蒙层 (Fading Mask)**: 当标签超出标签栏容器宽度时，在边缘显示的 40px 宽度的分层透明蒙层，用于提示可滚动。
- **选中态 (Active State)**: 活跃标签下方显示的 2px 高度亮色横条（建议色值：`#00FF80`）。
- **悬浮态 (Hover State)**: 鼠标指向按钮或标签时触发的微弱背景加亮效果。

## 3. 视窗布局 (Viewport)

- **终端视窗 (Terminal View)**: 占据窗口主体部分的黑色背景字符显示区。
- **状态栏 (Status Bar)**: 位于窗口最底部，提供连接状态、会话信息及系统时间的常驻显示。

# 专业名词

交互槽位 (Interaction Slot)：⚙ 获取焦点或悬浮后的那个背景区域，在我们的设计规范中正式命名为：“交互槽位 (Interaction Slot)” 或 “反馈底框 (Feedback Backplate)”。
