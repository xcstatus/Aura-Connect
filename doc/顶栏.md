# 顶栏
## 1. 顶栏区域 (Header Units)

- **顶栏 (Unified Header)**: 位于窗口最上方，高度固定为 **32px** 的全局交互容器。在 macOS 上采用沉浸式设计（透明底色、全尺寸内容视图）。
- **标签栏 (Tab Bar)**: 顶栏中用于容纳会话标签的水平区域，支持溢出滚动。
- **标签位 (Tab Item/Slot)**: 单个会话标签的物理区域，包含文字标题、背景 and 选中态指示条。默认宽度 140px，高度 32px。
- **操作组 (Action Group)**: 固定位置的功能按钮簇，包含 `+ (新建标签)` 和 `⚡ (快速连接)`。
- **控制组 (Control Group)**: 位于顶栏 right 侧，包含 `⚙ (设置)` 及非 Mac 平台的窗口控制按钮（最小化、最大化、关闭）。
- **物理分隔线 (Separator)**: 位于标签栏与操作组之间，高度为 14px-16px 的微弱垂直线，用于逻辑分层。

## 顶栏布局（从左至右）

红绿灯占位（固定左侧）
标签栏（占据红绿灯右侧剩余空间，背景色与中部终端区域一致；选中项在顶部增加一条线作为区分）
竖分割线（紧贴标签栏右侧）
操作组（含 ⚡ 与 + 按钮，紧贴分割线左侧，左对齐，背景色与顶栏背景色一致）
控制组（含 ⚙ 与窗口按钮，右对齐，位于顶栏最右侧，背景色与顶栏背景色一致）
布局行为

标签栏、分割线、操作组三者紧密相连，随标签数量变化整体平移；
标签栏内容增多时，标签栏区域水平滚动，分割线与操作组随之右移，保持相对位置不变；
操作组与控制组在视觉上分开，分别位于顶栏左侧功能区和右侧固定区。
样式说明

操作组、控制组与顶栏背景色保持一致，融入顶栏整体；
标签栏背景色与中部终端区域背景色一致，形成上下视觉连贯性；
标签选中时，仅在标签顶部显示一条线作为状态区分，不改变背景色。
