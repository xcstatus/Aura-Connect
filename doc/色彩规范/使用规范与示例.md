## 二、组件色彩对照表

### 2.1 按钮

> **Liquid Glass 按钮样式**：所有按钮使用 `liquid_glass.rs` 中的样式函数。

| 按钮类型 | 样式函数 | 背景色 | 文字色 | 边框 | 使用场景 |
|---------|---------|--------|--------|------|---------|
| Primary | `glass_primary_button_style()` | `accent_base` | `on_accent_label` | 无 | 主要操作：「连接」「保存」 |
| Secondary | `glass_secondary_button_style()` | `surface_1` | `text_primary` | `border_default` | 次要操作：「取消」「返回」 |
| Icon | `glass_icon_button_style()` | transparent→`glass_subtle` | `text_primary` | **无边框** | 工具栏、操作组 |
| Tab | `glass_tab_button_style()` | transparent→`glass_moderate` | `text_primary` | **无边框** | 标签栏切换 |
| Danger | `glass_danger_button_style()` | `error` | #FFFFFF | 无 | 危险操作：「删除」「断开」 |

> **核心原则**：图标按钮和标签按钮**无边框**，仅通过背景色透明度变化（`glass_subtle`/`glass_moderate`）暗示状态。

### 2.2 输入框

| 状态 | 背景 | 边框 | 说明 |
|------|------|------|------|
| Default | `surface_1` | `border_subtle` | 等待输入 |
| Focus | `surface_1` | `accent_base` (2px, `FOCUS_OUTLINE_ALPHA` 0.6) | 聚焦中 |
| Error | `surface_1` | `error` (2px) | 验证失败 |
| Disabled | `surface_2` | none | 不可编辑 |

### 2.3 卡片/面板

| 状态 | 背景 | 边框 | 说明 |
|------|------|------|------|
| Default | `surface_1` | `border_subtle` | 普通卡片 |
| Hover | `surface_2` | `border_subtle` | 可交互卡片（禁用边框突显） |
| Selected | `surface_1` | `accent_base` | 选中态 |

### 2.4 连接状态指示

| 状态 | 图标颜色 | 说明文字颜色 | 背景 |
|------|---------|-------------|------|
| 已连接 | `success` | `success` | 无 |
| 连接中 | `warning` | `warning` | 无 |
| 断开 | `text_disabled` | `text_disabled` | 无 |
| 错误 | `error` | `error` | 无 |

## 五、常见错误与纠正

### 5.1 错误示例

```rust
// ❌ 错误：在 Dark 主题下使用纯白色作为次要文字
label.style(TextStyle {
    color: Color::WHITE,  // 对比度过高
});

// ❌ 错误：硬编码颜色值
container.style(|_| container::Style {
    background: Some(Color::from_rgb8(0x1C, 0x1C, 0x1E).into()),
});

// ❌ 错误：给图标按钮加边框
button(icon)
    .style(|_| button::Style {
        border: Border { color: tokens.accent_base, width: 1.0, .. },
        ..Default::default()
    })

// ❌ 错误：使用不存在的 Token
let color = tokens.text_muted;       // ❌ 正确名: text_disabled
let bg = tokens.bg_surface;         // ❌ 正确名: bg_secondary
```

### 5.2 正确做法

```rust
// ✅ 正确：使用 DesignTokens
fn component(state: &IcedState) -> Element<'_, Message> {
    let tokens = tokens_for_state(state);
    container.style(move |_| container::Style {
        background: Some(tokens.bg_primary.into()),
        border_color: tokens.border_default,
        ..Default::default()
    })
}

// ✅ 正确：通过配色方案 ID 获取
let tokens = DesignTokens::for_color_scheme("terminal_dark");

// ✅ 正确：按钮状态使用对应的 Liquid Glass 样式函数
button(content)
    .style(glass_primary_button_style(tokens))
    // 或
    .style(glass_icon_button_style(tokens))

// ✅ 正确：图标按钮无边框
button(icon)
    .style(move |theme, status| button::Style {
        border: Border::default(), // ✅ 无边框
        background: match status {
            Status::Hovered => Some(tokens.glass_subtle.into()),
            _ => None,
        },
        ..Default::default()
    })

// ✅ 正确：错误状态使用 error 色
button.set_color(tokens.error);
```

## 六、主题切换注意事项

### 6.1 切换时需要重置的状态

```rust
// 当主题切换时，以下状态需要重新计算
struct ThemeTransition {
    // 1. 文本颜色
    all_text: Vec<Color>,

    // 2. 背景颜色
    all_backgrounds: Vec<Color>,

    // 3. 边框颜色
    all_borders: Vec<Color>,

    // 4. 交互状态（hover/active/focus）
    interaction_states: HashMap<WidgetId, WidgetState>,
}
```

### 6.2 过渡动画建议

```rust
// 主题切换使用平滑过渡
const THEME_TRANSITION_DURATION: Duration = Duration::from_millis(200);

fn animate_theme_change(from: ThemeId, to: ThemeId) {
    // 背景色：快速过渡
    lerp_color(from.bg_primary, to.bg_primary, anim_t(200ms));

    // 强调色：稍慢，突出变化
    lerp_color(from.accent_base, to.accent_base, anim_t(250ms));
}
```
