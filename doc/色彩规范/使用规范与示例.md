## 二、组件色彩对照表

### 2.1 按钮

| 按钮类型 | 背景色 | 文字色 | 边框色 | 使用场景 |
|---------|--------|--------|--------|---------|
| Primary | `accent_base` | `on_accent_label` | none | 主要操作：「连接」「保存」 |
| Secondary | `surface_1` | `text_primary` | `border_default` | 次要操作：「取消」「返回」 |
| Ghost | transparent | `text_secondary` | none | 辅助操作：「了解更多」 |
| Danger | `error` | `on_accent_label` (#FFF) | none | 危险操作：「删除」「断开」 |

### 2.2 输入框

| 状态 | 背景 | 边框 | 说明 |
|------|------|------|------|
| Default | `surface_1` | `border_subtle` | 等待输入 |
| Focus | `surface_1` | `accent_base` (2px) | 聚焦中 |
| Error | `surface_1` | `error` (2px) | 验证失败 |
| Disabled | `surface_2` | none | 不可编辑 |

### 2.3 卡片/面板

| 状态 | 背景 | 边框 | 说明 |
|------|------|------|------|
| Default | `surface_1` | `border_subtle` | 普通卡片 |
| Hover | `surface_2` | `border_default` | 可交互卡片 |
| Selected | `surface_1` | `border_default` + accent | 选中态 |

### 2.4 连接状态指示

| 状态 | 图标颜色 | 说明文字颜色 | 背景 |
|------|---------|-------------|------|
| 已连接 | `success` | `success` | 无 |
| 连接中 | `warning` | `warning` | 无 |
| 断开 | `text_disabled` | `text_disabled` | 无 |
| 错误 | `error` | `error` | 无 |

## 五、常见错误与纠正

### 5.1 错误示例

```rust
// ❌ 错误：在 Dark 主题下使用纯白色作为次要文字
label.style(TextStyle {
    color: Color::WHITE,  // 对比度过高
});

// ❌ 错误：硬编码颜色值
container.style("background: #1C1C1E");

// ❌ 错误：在错误状态使用 success 色
button.set_color(Color::from_rgb8(0x00, 0xFF, 0x80)); // 绿色不是错误色
```

### 5.2 正确做法

```rust
// ✅ 正确：使用 DesignTokens
let tokens = DesignTokens::for_id(current_theme);
label.style(TextStyle {
    color: tokens.text_secondary,  // 语义化引用
});

// ✅ 正确：使用主题感知的颜色获取
fn get_button_color(state: ButtonState) -> Color {
    match state {
        State::Primary => tokens.accent_base,
        State::Hover => tokens.accent_hover,
        State::Active => tokens.accent_active,
    }
}

// ✅ 正确：错误状态使用 error 色
button.set_color(tokens.error);
```

## 六、主题切换注意事项

### 6.1 切换时需要重置的状态

```rust
// 当主题切换时，以下状态需要重新计算
struct ThemeTransition {
    // 1. 文本颜色
    all_text: Vec<Color>,

    // 2. 背景颜色
    all_backgrounds: Vec<Color>,

    // 3. 边框颜色
    all_borders: Vec<Color>,

    // 4. 交互状态（hover/active/focus）
    interaction_states: HashMap<WidgetId, WidgetState>,
}
```

### 6.2 过渡动画建议

```rust
// 主题切换使用平滑过渡
const THEME_TRANSITION_DURATION: Duration = Duration::from_millis(200);

fn animate_theme_change(from: ThemeId, to: ThemeId) {
    // 背景色：快速过渡
    lerp_color(from.bg_primary, to.bg_primary, anim_t(200ms));

    // 强调色：稍慢，突出变化
    lerp_color(from.accent_base, to.accent_base, anim_t(250ms));
}
```
