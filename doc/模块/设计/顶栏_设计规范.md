# 顶栏

## 1. 顶栏区域 (Header Units)

- **顶栏 (Unified Header)**: 位于窗口最上方，高度固定为 **32px** 的全局交互容器。在 macOS 上采用沉浸式设计（透明底色、全尺寸内容视图）。
- **标签栏 (Tab Bar)**: 顶栏中用于容纳会话标签的水平区域，标签位等宽分布，自适应填满可用空间。
- **标签位 (Tab Item/Slot)**: 单个会话标签的物理区域，包含文字标题、背景和选中态指示条。默认宽度 140px，高度 32px。标签获得焦点时显示关闭按钮。标签宽度不足时文字自动截断显示省略号。
- **操作组 (Action Group)**: 固定位置的功能按钮簇，包含 `+ (新建标签)` 和 `⚡ (快速连接)`。无背景色，按钮尺寸 36x36px。
- **控制组 (Control Group)**: 位于顶栏 right 侧，包含 `⚙ (设置)` 及非 Mac 平台的窗口控制按钮（最小化、最大化、关闭）。无背景色，按钮尺寸 36x36px。

## 顶栏布局（从左至右）

```
[红绿灯占位 (76px)] [标签栏 (自适应)] [操作组] [控制组]
```

- 红绿灯占位：macOS 固定 76px（用于放置窗口控制按钮）；非 macOS 为 0
- 标签栏：占据剩余空间，标签位等宽分布，背景色与终端区域一致
- 操作组：左对齐，无背景色，紧贴标签栏
- 控制组：右对齐，无背景色，位于顶栏最右侧

## 布局公式

```
标签栏宽度 = window_width - 红绿灯宽度 - 操作组宽度 - 控制组宽度
单个标签宽度 = 标签栏宽度 / 标签数量
```

## 尺寸常量


| 常量                   | 值                              | 说明               |
| -------------------- | ------------------------------ | ---------------- |
| TOP_BAR_H            | 32px                           | 顶栏高度             |
| TOP_ICON_BTN         | 36px                           | 按钮尺寸（操作组/控制组）    |
| TRAFFIC_LIGHT_BAND_W | 76px (macOS) / 0 (非 macOS)     | 红绿灯占位宽度          |
| TAB_CHIP_WIDTH       | 140px                          | 标签默认宽度（用于单个标签时）  |
| TAB_ACTION_GROUP_W   | 78px                           | 操作组宽度（2个按钮 + 间距） |
| TOP_CONTROL_GROUP_W  | 36px (macOS) / 114px (非 macOS) | 控制组宽度            |


## 视觉效果


| 区域   | 样式                    |
| ---- | --------------------- |
| 顶栏背景 | `bg_primary`          |
| 标签栏  | `bg_primary` 背景（主题跟随） |
| 操作组  | 无背景色                  |
| 控制组  | 无背景色                  |
| 红绿灯  | macOS 原生              |


## 标签选中态

- 标签选中时，在标签顶部显示一条 2px 高的 `accent_base` 色指示线
- 标签获得焦点时，显示关闭按钮 `×`
