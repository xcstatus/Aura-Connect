# 终端配色方案

## 一、概述

终端配色方案功能允许用户自定义终端的颜色显示，支持预设方案选择（层次 1）和快速调整（层次 2）。

### 1.1 功能特性

| 特性 | 描述 |
|------|------|
| 预设方案 | 7 种内置配色方案可选 |
| 终端主题 | 支持跟随 UI 主题 / 强制深色 / 强制浅色 |
| 快速调整 | 支持自定义背景色、前景色、光标色 |
| 实时预览 | 设置界面提供颜色预览 |
| 持久化 | 所有设置自动保存到配置文件 |

pub struct TerminalPalette {

## 三、用户界面

### 3.1 布局结构

终端设置页面包含以下子页签：

```
终端设置
├── 渲染 (sub=0)
├── 配色 (sub=1)      ← 预设方案选择
├── 文字 (sub=2)
├── 快速调整 (sub=3)  ← 自定义颜色
└── 交互 (sub=4)
```

### 3.2 配色页签 (sub=1)

#### 预设方案选择

- **布局**: 网格布局，每行显示 2 个方案
- **每项包含**:
  - 配色预览框（60x40px）
  - 方案名称
  - 选中状态指示（"已应用" 文字 + 边框高亮）

#### 终端主题选择

- 选项: 跟随系统 / 深色 / 浅色
- 选中状态: 文字高亮 + 勾选标记

### 3.3 快速调整页签 (sub=3)

#### 自定义颜色输入

| 字段 | 标签 | 说明 |
|------|------|------|
| 背景色 | Background | HEX 格式，如 #RRGGBB |
| 前景色 | Foreground | HEX 格式，如 #RRGGBB |
| 光标色 | Cursor | HEX 格式，如 #RRGGBB |

#### 重置按钮

- 功能: 清除所有自定义颜色，恢复到预设值
- 样式: 红色背景，白色文字

#### 实时预览

- 显示带有 ANSI 彩色的示例终端输出
- 实时反映用户输入的自定义颜色

## 四、技术实现

### 4.1 配色应用流程

```
用户选择配色方案
        ↓
SettingsFieldChanged(ColorScheme)
        ↓
apply_settings_field()
        ↓
TerminalController::apply_terminal_palette_for_scheme()
        ↓
palette::scheme_vt_bytes() → VT 字节序列
        ↓
ghostty_vt::write_vt() → 发送到终端
        ↓
terminal.refresh_terminal_snapshots() → 刷新渲染
```

### 4.2 VT 调色板

定义在 `src/terminal/palette.rs`，通过 xterm OSC 4 协议设置 ANSI 颜色：

```rust

```
